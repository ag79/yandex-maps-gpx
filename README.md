# Делаем gpx файл из маршрута Яндекс Карт

Создаем gpx файл с нужными объектами:

- маршрутами
- треками
- путевыми точками

Добавляем в файл высотные данные из [http://www2.jpl.nasa.gov/srtm/](SRTM).

Используем gpxpy, SRTM.py.
